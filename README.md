# Ork CSV

Ork CSV is a library for reading and writing CSV files.

## Requirements
* PHP 7.1

## Installation

### Via command line:
```bash
$ composer require ork/csv
```

### Via composer.json:
```json
"require": {
    "ork/csv": "*"
},
```

## Documentation

See the [wiki](https://github.com/AlexHowansky/ork-csv/wiki).

## Development

### Testing
```bash
$ composer test
```

### Generate API Documentation
```bash
$ composer doc
```

### Code Styling Validation
```bash
$ composer phpcs
```
