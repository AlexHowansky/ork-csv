# Ork CSV

Ork CSV is a library for reading and writing CSV files.

## Requirements
* PHP 7.1

## Installation

## Via command line:
```bash
$ composer require ork/csv
```

## Via composer.json:
```json
"require": {
    "ork/csv": "*"
},
```

# Generate Documentation
```bash
$ composer doc
```

# Testing
```bash
$ composer test
```

# Code Styling Validation
```bash
$ composer phpcs
```
